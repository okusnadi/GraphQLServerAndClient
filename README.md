# ASP.NET Core GraphQL server and angular GraphQL client

ASP.NET Core GraphQL server and angular GraphQL client. 

## Run
Open solution file **GraphQLServerAndClient.sln** with Visual Studio 2017, and press F5, and then open browser at **localhost:62001**.
